# BrokersEdge TMS Website

Static landing page built with HTML, CSS, JavaScript, and local image assets.

## How to run locally
Open `index.html` in your browser.

## GitHub Pages setup
1. Upload the full folder contents to your GitHub repository.
2. Keep the `/assets` folder at the same level as `index.html`.
3. Go to Repository Settings > Pages.
4. Deploy from the main branch root.

## Why images will not break
All image paths are relative:
- `assets/brokersedge-logo-wide.png`
- `assets/brokersedge-logo-mark.png`
- `assets/dashboard-ui.png`
- `assets/load-board-ui.png`

Do not rename or move these files unless you update the paths in `index.html`.
